## CLIP 5 — TRIBE V2: EL CEREBRO DESPIERTA
**Duración:** 0:29 - 0:37 (8s)

### Video Direction
**TRANSICIÓN A OTRO MUNDO.** El screen recording del prompt (del clip 3) reaparece brevemente y SE DISUELVE en una visualización 3D de un cerebro humano estilo TRIBE V2 — cortical surface mesh, fsaverage5. El cerebro empieza gris, oscuro. Mientras se escucha la narración, las regiones se encienden: corteza visual en azul, red de lenguaje en verde, corteza prefrontal en dorado. Colores de heat map expandiéndose en tiempo real. El cerebro rota lentamente. Colores pulsan como actividad neuronal. Data overlays aparecen: porcentajes de activación, labels de regiones. Debe verse como OBSERVAR el pensamiento sucediendo.

### TTS Narration
```
Esto es lo que pasa en el cerebro de tu audiencia cuando ve contenido creado con este método. Cada zona que se enciende es una razón para compartir.
```

### Music
**NUEVO MUNDO SONORO.** Desde el silencio del clip 4, entra un sintetizador — no agresivo, sino ALIENÍGENA. Como escuchar el interior de una computadora pensar. Texturas granulares, pads cristalinos, un sweep de frecuencia ascendente lento. Referencia: Ryuichi Sakamoto meets Arca. Clicks pequeños y pops como sinapsis. Sub-bass profundo pulsando a 60 BPM — el ritmo del cerebro. NO piano. Esto es sonido de CIENCIA, no de emoción. 8 segundos de asombro sonoro puro.

### Overlay Text
- 0:30 — "El cerebro de tu audiencia" (blanco, aparece sobre la visualización)
- 0:34 — "en tiempo real." (dorado, debajo)

### Notas de Edición
- ESTE ES EL PRIMER WOW MOMENT. La transición de la historia personal (clips 1-4) al mundo de la ciencia/datos debe sentirse como cambiar de dimensión.
- El cerebro 3D DEBE verse premium. Si la calidad de VFX no es suficiente, es mejor usar footage estilo neuroscience documental que una animación barata.
- Las regiones del cerebro deben encenderse secuencialmente, no todas a la vez. Crear narrativa visual.
- El contraste entre el silencio del clip 4 y este mundo de sonido crea un efecto de "despertar".

### Notas de Edición

- La sincronización entre radar y cerebro debe ser PERFECTA — cada dimensión que sube se mapea a una región específica. No puede ser random.
